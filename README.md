## Suggest Project

1. その規格のブランチを作成します。
1. 企画書を"/project/"の中に作成します。
1. pull requestを送ります。
1. 作成するかの判断をします。
1. 作成するのであればマージ、そうでなければ閉じるか保留にしておく。
1. そのプロジェクトのリポジトリを作成します。
1. プロジェクト開始。
