## Suggest Project

1. その企画のブランチを作成します。
3. 企画書を"/project/"の中に作成します。
4. pull requestを送ります。
5. 作成するかの判断をします。
6. 作成するのであればマージ、そうでなければ閉じるか保留にしておく。
7. そのプロジェクトのリポジトリを作成します。
8. プロジェクト開始。
